TF Easy Colors Map 2.x - QUICK GUIDE


QUICK REFERENCE
LEFT click   Set the foreground color
LEFT click + SHIFT   Set the background color
LEFT click + CTRL   Change the name of the clicked Color or Group Title
RIGHT click   Add a new color, at the end of the Map
RIGHT click + SHIFT   Add a new color after the clicked Color or Group Title
RIGHT click + CTRL   Show a popup menu with some functionalities



Create or load a Colors Map
Press the 'Open Map' button. Type a new file name to create a new Colors Map or select and open an existing Colors Map.
Note that a Colors Map file is just an ordinary text file with .cmap extension.

Add a new color
With a Krita document opened and the desired color set as foreground color, just right click on the Map panel. Type a short name for the color and press [OK] (if you press [CANCEL] no color is added).

Add a new color in a specific position
Follow the instructions above, but right click + SHIFT button on an existing color. The new color will be added after that color.
Note that you can click a Group Title to add the color as first color.

Add new colors automatically
Press the 'Auto Add Colors' button. Every time you change the foreground color, the new color is automatically added to the Map. Press again the 'Auto Add Colors' button to stop this functionality.
Note that the new colors are added at the end of the Map.

Select a Color of the Map
Left click an existing color: it will set the Krita document foreground color.
Left click + SHITF button an existing color: it will set the Krita document background color.

Colors Map Management
Right click + CTRL button an existing Color or Group Title: a popup menu will apear with the available functionalities.

- Add Group Title: add a new Group Title after the clicked Color or Group.
- Rename: change the name of the clicked Color or Group (you can also Left click + CTRL the item to rename).
- Cut: select a Color or a Group for a successive 'paste' operation.
- Paste: move a Color or a Group Title after the clicked color (you must click a Color).
- Paste Group: move a Group Title and all the Colors under it after the clicked color (you must click a Color).
- Delete: delete the clicked Color or Group (the Group Title only will be deleted, not the Colors under it). - Delete Group: delete the clicked Group and all the Colors under it.

Temporary Colors (Secondary palette)
If you want to collect some color, but you don't want to save them into your Colors Map, you can add colors to the palette under the Colors Map. Just right click to add colors and left click for setting the foreground color (+SHIFT for setting the background instead).

Settings
Press the 'Settings' button. A window will appear showing supported settings.
- Set Color size: allow to change the size of the Color boxes.
- Set Color Name size: allow to change the size of the name inside the Color boxes. - Show / Hide Groups: allow to show or hide Groups of Colors in the Colors Map view.



Colors Map Editor
Press the 'Map Editor' button. A window will appear showing the content of the Colors Map file. You can manually perform modifications and save the Map.
Use this functionality with caution.
